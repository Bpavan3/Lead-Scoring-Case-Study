# Lead Scoring Case Study

This project focuses on building a lead scoring model using Logistic Regression
to identify high-potential leads and improve conversion rates.

## Business Objective
- Predict probability of lead conversion
- Identify key drivers of conversion
- Optimize sales calling strategy

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection using VIF and p-values
- Logistic Regression model
- Business-driven cutoff strategy
